Info
What is VennV anticheat? VennV is an advanced packet-based minecraft-bedrock anticheat, made to perform at any player count, on any server version between 1.12.x and 1.19.x, without any performance overhead. VennV includes an extremely useful API that allows you to send packets to the server running VennV and get the results that have been checked by VennV. However they must be valid packages and are not intended to intentionally sabotage VennV.
